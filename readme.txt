Clean admin color theme.

== Installation ==
1. Unzip the downloaded .zip file.
2. Upload the plugin folder to the wp-content/plugins/ directory of your WordPress site.
3. Go to the 'Plugins' menu in WordPress and activate the Admin UI - Preset color schemes.
